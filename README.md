# verziókezelés

Parancssorban PAT-al:

-Helyi repó inicializálása:
    -**git init**

-Projekt könyvtár ellenörzése:
    -**git status**

-Ha van elmenteni való akkor előkészítjük a színpadra(stage):
    -**git add .**

-Elmentjük a helyi repóba(commit):
    -**git commit -m "first commit"**

-Ellenörzés:
    -**git status**
